This week's task was to create six different arduino combinations.



SCANNING


1-2) This project is blinking green LED: it blinks in turn. 3) Blue LED: it turns on and off with the user-set time interval. 4) transitor: it starts working when it is plugged to the computer 5) sound: the arduino starts playing a song 6) temperature: arduino even detects temperature? What CAN'T the arduino do?! 7) Motor: the motor moves back and forth.




Reflection


When I first got the kit, I was overwhelmed. However, I got to learn more about the arduino and started having fun as Mr. Gerber and my classmates guided me throughout the process. I was really intrigued in knowing what the arduino could do, because it seemed like there was nothing impossible for it. When I get a chance next time, I would like to try out more arduino experiments!
